# 🎙️ ElevenLabs API 설정 가이드

AutoVideo 프로젝트에서 고품질 AI 음성을 생성하기 위해 ElevenLabs API 키가 필요합니다. 아래 절차를 따라 키를 발급받고 설정해 주세요.

## 1. API 키 발급 받기
1. [ElevenLabs 웹사이트](https://elevenlabs.io/)에 접속하여 로그인합니다.
2. 우측 상단 프로필 아이콘을 클릭하고 **Profile + API key** 메뉴를 선택합니다.
3. **API Key** 섹션에서 눈 모양 아이콘을 클릭하여 키를 확인하거나, **Create New Key**가 있다면 새로 생성합니다.
4. `sk_...` 로 시작하는 키 값을 복사합니다.

## 2. 프로젝트에 키 적용하기
프로젝트 루트 디렉토리의 `.env.local` 파일을 열고 아래와 같이 입력합니다.

```bash
# .env.local 파일 (없으면 생성)

# 기존 설정들...

# ElevenLabs API Key 추가
ELEVENLABS_API_KEY=여기에_복사한_키를_붙여넣으세요
```

> **주의**: `sk_`를 포함한 전체 문자열을 입력해야 합니다. 따옴표는 없어도 됩니다.

## 3. 적용 확인
1. 파일을 저장합니다.
2. 실행 중인 터미널에서 개발 서버를 재시작합니다.
   - 단축키: `Ctrl + C` 로 중지 후 `npm run dev` 입력
3. **Voice Studio** 메뉴 또는 **AutoVideo** 생성 과정에서 음성 생성이 정상적으로 작동하는지 확인합니다.

## 참고 사항
- 무료 계정의 경우 글자 수 제한이 있을 수 있습니다.
- API 키는 타인에게 노출되지 않도록 주의해 주세요. (GitHub 등에 커밋 금지)
