# 🎬 AutoVideo 사용 가이드북

> **누구나 쉽게 AI 영상을 만들 수 있어요!**
>
> 이 가이드북은 코딩을 전혀 몰라도 AutoVideo를 사용할 수 있도록 친절하게 설명해 드립니다.

---

## 🌟 AutoVideo가 뭔가요?

AutoVideo는 **주제만 입력하면 AI가 알아서 영상을 만들어주는** 마법 같은 도구예요!

예를 들어, "오늘의 비트코인 시세 분석"이라고 입력하면:
1. ✍️ AI가 대본을 써주고
2. 🎙️ 전문 성우 목소리로 읽어주고
3. 🖼️ 예쁜 이미지를 그려주고
4. 🎥 이미지에 움직임을 넣어 영상으로 만들어줘요!

---

## 🚀 시작하기 전에

### 필요한 것들

| 항목 | 설명 | 어디서 구하나요? |
|------|------|------------------|
| 💻 컴퓨터 | 영상 생성을 위한 PC | 있으시죠? 😊 |
| 🌐 인터넷 | 안정적인 인터넷 연결 | |
| 🎮 ComfyUI | 영상 생성 프로그램 (무료!) | 별도 설치 필요 |

> 💡 **팁:** ComfyUI 설치는 처음에 복잡할 수 있어요. 개발자분께 도움을 요청하세요!

---

## 📖 사용 방법 (Step by Step)

### Step 1️⃣ 새 프로젝트 만들기

1. 웹 브라우저에서 `http://localhost:3000` 주소로 접속해요
2. 메인 화면에서 **"새 프로젝트"** 버튼을 클릭해요
3. 다음 정보를 입력해요:

   | 항목 | 예시 | 설명 |
   |------|------|------|
   | **제목** | 비트코인 분석 영상 | 프로젝트 이름이에요 |
   | **주제** | 오늘의 비트코인 시세와 전망 | AI가 이걸 보고 대본을 써요 |
   | **길이** | 60초 | 영상 길이예요 |
   | **스타일** | economy-1 | 영상의 비주얼 느낌이에요 |

4. **"생성"** 버튼을 누르면 끝!

---

### Step 2️⃣ 스크립트(대본) 생성하기

프로젝트를 만들면 자동으로 **스크립트 페이지**로 이동해요.

1. **"스크립트 생성"** 버튼을 클릭해요
2. ⏳ 잠시 기다리면... AI가 대본을 완성해요! (보통 10~15초)
3. 생성된 대본이 마음에 안 드시면 직접 수정도 가능해요!

> 📝 **대본 수정 팁:**
> - 각 컷(장면)별로 텍스트를 수정할 수 있어요
> - 너무 긴 문장은 짧게 나누는 게 좋아요
> - 어려운 단어는 쉬운 말로 바꿔보세요

---

### Step 3️⃣ 음성 선택하기

이제 대본을 읽어줄 **AI 성우**를 선택해요!

1. "다음 단계" 버튼을 눌러 **음성 페이지**로 이동
2. 여러 목소리 중에서 마음에 드는 것을 선택해요
3. 🎧 미리듣기로 목소리를 확인할 수 있어요
4. **"음성 생성"** 버튼을 클릭!

> 🎙️ **음성 종류:**
> - 남성 목소리 (Adam, Antoni 등)
> - 여성 목소리 (Bella, Elli 등)
> - 각 목소리마다 톤과 분위기가 달라요!

---

### Step 4️⃣ 이미지 생성하기

AI가 각 장면에 맞는 **그림**을 그려줘요!

1. "다음 단계" 버튼을 눌러 **이미지 페이지**로 이동
2. 각 컷마다 **"이미지 생성"** 버튼을 클릭
3. ⏳ AI가 그림을 그리는 동안 잠시 기다려요 (약 10~15초)
4. 생성된 이미지가 마음에 안 들면 다시 생성할 수 있어요!

> 🎨 **이미지 팁:**
> - 처음 생성된 이미지의 캐릭터가 다른 컷에서도 비슷하게 유지돼요
> - 스타일(economy-1, anime 등)에 따라 그림체가 달라져요

---

### Step 5️⃣ 영상 클립 생성하기 ⭐

가장 중요한 단계! 이미지에 **움직임**을 넣어 영상으로 만들어요.

1. "다음 단계" 버튼을 눌러 **비디오 페이지**로 이동
2. 워크플로우를 선택해요:
   
   | 워크플로우 | 특징 |
   |------------|------|
   | **Rapid AIO Mega + Sage v2** | ⭐ 추천! 빠르고 품질 좋음 |
   | Rapid AIO Mega + Sage | 안정적인 품질 |
   | LF i2v (Batch) v1.1 | 사실적인 느낌 |
   | LTX Video (Fast) | 가장 빠름 (품질↓) |

3. **"전체 생성 시작"** 버튼을 클릭!
4. ⏳ 영상 생성에는 시간이 좀 걸려요 (컷당 1~2분)
5. 오른쪽의 **실시간 로그**에서 진행 상황을 확인할 수 있어요

> ⚠️ **주의사항:**
> - 영상 생성 중에는 컴퓨터를 켜두세요!
> - ComfyUI가 반드시 실행 중이어야 해요

---

### Step 6️⃣ 미리보기 & 렌더링

모든 영상 클립이 완성되면 **최종 영상**을 만들어요!

1. "다음 단계" 버튼을 눌러 **미리보기 페이지**로 이동
2. 미리보기 영상을 재생해서 확인해요
3. 마음에 들면 **"렌더링"** 버튼을 클릭
4. 렌더링이 완료되면 **"다운로드"** 버튼으로 영상을 저장해요!

🎉 **축하해요! 영상이 완성됐어요!**

## 💡 자주 묻는 질문 (FAQ)

### Q: 영상 생성이 안 돼요!
> **A:** ComfyUI가 실행 중인지 확인해 주세요. 
> 브라우저에서 `http://127.0.0.1:8188`로 접속해서 ComfyUI 화면이 보이면 정상이에요.

### Q: 이미지가 이상하게 생성돼요
> **A:** AI가 가끔 실수할 수 있어요. "다시 생성" 버튼으로 새로운 이미지를 받아보세요!

### Q: 음성이 부자연스러워요
> **A:** 다른 음성을 선택해 보세요. 한국어는 "Rachel"이나 "Antoni" 같은 음성이 자연스러운 편이에요.

### Q: 영상 생성이 너무 오래 걸려요
> **A:** 고사양 그래픽 카드가 있으면 더 빨라요. "LTX Video (Fast)" 워크플로우를 선택하면 속도가 빨라지지만 품질은 조금 낮아질 수 있어요.

### Q: 특정 컷만 다시 생성하고 싶어요
> **A:** 해당 컷을 클릭한 뒤 "현재 컷 생성하기" 버튼을 누르면 그 컷만 다시 생성돼요!
